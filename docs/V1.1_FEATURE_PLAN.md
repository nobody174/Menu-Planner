# Pi-Menu v1.1 Feature Plan

**Status:** Ready for Implementation  
**Target Release:** Q3 2026  
**Focus:** Recipe Discovery, Preloaded Content, Personal Recipe Arsenal  

---

## Overview

v1.1 transforms Pi-Menu from a blank-slate app into a feature-rich system with:
- **Instant recipe library** for new users (75+ bilingual recipes)
- **Easy recipe discovery** through curated packs
- **Personal recipe management** (export/import/backup)
- **Community-friendly** recipe sharing

---

## Feature Summary

### 🍽️ Recipe Discovery & Preloaded Content

#### 1. Free Recipe Sources Library ⚡
**Quick Win** — Documentation only, no code changes

**What:** `FREE_RECIPE_SOURCES.md` - Curated list of legitimate recipe sources
- 10+ Norwegian free recipe sources
- 10+ English free recipe sources
- Usage rights information
- Manual import instructions

**Impact:** Users know where to find recipes to manually add
**Effort:** 1-2 days (research & documentation)

---

#### 2. Preloaded Recipe Packs System 📦
**Foundation** — Core data + UI enhancement

**What:** Optional recipe collections users can import with one click

**Packs Created:**
1. **Popular Norwegian Recipes** (15 recipes)
   - Fårikål, Raspeball, Kjøttkaker, etc.
   - Traditional Norwegian favorites

2. **European Classics** (15 recipes)
   - Italian, French, Spanish dishes
   - Pasta, Risotto, Bouillabaisse, etc.

3. **Nordic Classics** (15 recipes)
   - Swedish Meatballs, Danish specialties
   - Scandinavian heritage

4. **Holiday Recipes** (12 recipes)
   - Christmas, Easter, special occasions
   - Seasonal festive meals

5. **Summer Recipes** (15 recipes)
   - Fresh Salads (5)
   - Grill Recipes (5)
   - Seafood - Shrimp, Crab, Fish (5)

**Total:** 72+ new bilingual recipes (Norwegian + English)

**Recipe Structure:**
- Title (NO/EN)
- Subtitle (NO/EN)
- Ingredients with quantities
- Instructions (NO/EN)
- Category mapping
- Tags (seasonal, difficulty, time)
- Allergen info

**UI Changes:**
- "Import Recipe Packs" button in Settings
- Pack selection modal
- One-click import
- "Manage Imported Packs" interface

**Impact:** New users get immediate recipe library to explore
**Effort:** 4-5 days (data creation + UI)

---

#### 3. Personal Recipe Arsenal Management 📚
**Advanced Feature** — Export/import + collection management

**What:** Users can organize and share their own recipe collections

**Features:**
- Export personal recipes as JSON pack
- Import custom recipe packs
- Create named personal collections
- Backup/restore collections
- Share templates with family

**Impact:** Enable power users and community sharing
**Effort:** 2-3 days (backend + UI)

## Technical Notes

### Architecture
- Recipe packs stored as JSON in `data/recipe-packs/`
- Schema validation on import
- No database changes required
- Backward compatible with existing recipes

### API Endpoints (New)
```
POST /api/recipe-packs/list      # Get available packs
POST /api/recipe-packs/import    # Import a pack
POST /api/recipes/export         # Export personal recipes
POST /api/recipes/import         # Import custom pack
GET  /api/collections            # List saved collections
```

### UI Components
- Settings page new section: "Recipe Packs"
- Modal: "Select Packs to Import"
- Modal: "Manage Your Collections"
- Button: "Export My Recipes"

## Risk Mitigation

| Risk | Mitigation |
|------|-----------|
| **Low engagement with packs** | Start with popular, tested recipes |
| **Translation quality issues** | Native speakers review all NO/EN |
| **Data structure problems** | Strict schema validation |
| **Import conflicts** | Gentle duplicate handling (rename) |
| **Performance with 100+ recipes** | Database-ready design (no current issues) |
