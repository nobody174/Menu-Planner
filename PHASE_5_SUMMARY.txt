================================================================================
ARCHITECTURE
================================================================================

FLOW:
  User checks items -> Clicks "Send Handleliste til telefon"
       ↓
  JavaScript collects checked items
       ↓
  POST /api/sync-shopping-list (with items)
       ↓
  Flask backend calls ToDoSync.sync_shopping_list_to_todo()
       ↓
  ToDoSync authenticates with Microsoft Graph
       ↓
  Creates/updates tasks in "Handleliste" list
       ↓
  Returns success to UI
       ↓
  User sees confirmation message
       ↓
  Opens To Do app on phone to see items

FUTURE (Not yet implemented):
  User checks items in To Do app
       ↓
  Pi-Menu "Sync with phone" button

  Fetches completed items from To Do

  Updates UI to show which items already bought
